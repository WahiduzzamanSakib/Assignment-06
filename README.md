# Assignment-06

## 📌 Project Overview
**Assignment-06** is a **React-based interactive shopping cart application** that allows users to browse products, add them to the cart, and experience a dynamic, responsive interface. The project leverages modern frontend technologies to provide real-time feedback and a smooth shopping experience.

---

## 🛠️ Technologies Used
- HTML5
- React.js 
- Tailwind CSS 
- DaisyUI
- React-Toastify– Toast notifications for user actions  
- JSON – Stores product data  

---

## ✨ Features
- **Interactive Shopping Cart** – Click a product to add it automatically to the cart.  
- **Dynamic Navbar Count** – The cart count updates in real-time when products are added.  
- **Proceed to Checkout** – Clears the cart for a seamless checkout experience.  

### Optional UX Enhancements
- **Cart Interaction** – Clicking products increases the cart count; checkout clears it.  
- **Buy Now Feedback** – Clicking "Buy Now" displays an “Added to cart” toast notification.

## 🚀 How to Run Locally
1. **Clone the repository**  
   ```bash
   git clone https://github.com/WahiduzzamanSakib/Assignment-06.git
````

2. Navigate to the project folder

   ```bash
   cd Assignment-06
   ```
3. Install dependencies

   ```bash
   npm install
   ```
4. Start the development server

   ```bash
   npm start
   ```
5. Open in Browser
